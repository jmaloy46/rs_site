# Papa Shaps Memorial Website

A beautiful tribute website featuring floating background text and interactive family photos arranged in a circle layout.

## Features

- Black background with floating white text (father's letter)
- Circular photo layout with smooth animations
- Hover and click interactions to enlarge photos
- Mobile responsive design
- Local image and text storage
- Optimized for GitHub Pages deployment

## Setup

1. Place family photos in the `images/` directory (photo1.jpg through photo8.jpg)
2. Replace the content in `lorem_ipsum.txt` with the actual letter
3. Commit and push to GitHub
4. Enable GitHub Pages in repository settings
5. Configure custom domain (papashaps.com) in GitHub Pages settings

## File Structure

```
├── index.html          # Main HTML file
├── styles.css          # All styling and animations
├── script.js           # JavaScript functionality
├── lorem_ipsum.txt     # Background text content
├── images/             # Photo storage directory
├── CNAME              # Custom domain configuration
└── README.md          # This file
```

## Local Development

Simply open `index.html` in a web browser to view the site locally.

## Deployment

The site is configured for GitHub Pages deployment with the custom domain papashaps.com.
